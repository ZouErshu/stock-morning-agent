## 阶段 ①：安装 Python 环境

### 1.1 检查是否已安装 Python

按 `Win + R`，输入 `cmd` 回车，打开命令提示符，输入：

```
python --version
```

- 如果显示 `Python 3.10.x` 或更高版本 → ✅ 已安装，跳到阶段②
- 如果显示"不是内部或外部命令" → 继续下面步骤安装

### 1.2 下载安装 Python

1. 访问官网：https://www.python.org/downloads/
2. 点击黄色大按钮 `Download Python 3.12.x`
3. 双击下载的安装包，**重要**：
   - ✅ **勾选底部 "Add Python to PATH"**（这一步非常关键！）
   - 然后点击 `Install Now`
4. 等待安装完成，点击 `Close`

### 1.3 验证安装

重新打开 cmd（关闭之前的再开），输入：

```
python --version
pip --version
```

两个都有版本号输出就成功了。

## 阶段 ③：申请 DeepSeek API Key

DeepSeek 是给早报提供"AI大脑"的服务，国内直连，价格极低（每天1毛钱）。

### 3.1 注册账号

1. 访问 https://platform.deepseek.com/
2. 用手机号注册并登录

### 3.2 创建 API Key

1. 登录后，左侧菜单点击 **「API Keys」**
2. 点击 **「创建 API Key」**
3. 随便起个名字，比如 `stock-morning`
4. 点击 **「创建」**
5. **立刻复制生成的 Key**（格式类似 `sk-abc123def456...`），保存到记事本

> ⚠️ Key 只显示一次，关掉就看不到了！如果忘了就重新创建一个。

### 3.3 充值（10元够用1个月）

1. 左侧菜单点击 **「充值」**
2. 充 10 元即可（微信/支付宝扫码）
3. DeepSeek 定价：输入 1元/百万token，输出 2元/百万token，每天早报约用 0.03元

### 3.4 保存你的 Key

把 Key 记录下来，等下要填到配置文件里：

```
我的 DeepSeek API Key: sk-________________________________________
```

## 阶段 ④：创建飞书机器人（最关键的一步）

这一步稍复杂，请仔细跟着做。飞书机器人有两个作用：①定时推送早报给你 ②你@它可以直接对话。

### 4.1 进入飞书开放平台

1. 用电脑浏览器访问：https://open.feishu.cn/app
2. 用你的飞书账号扫码登录

### 4.2 创建企业自建应用

1. 点击 **「创建企业自建应用」**
2. 填写：
   - 应用名称：`市场早报助手`（随便起）
   - 应用描述：`每日市场早报推送`
   - 应用图标：随便传一个图或用默认
3. 点击 **「创建」**

### 4.3 拿到 App ID 和 App Secret

创建后进入应用详情页，在 **「凭证与基础信息」** 里能看到：
- **App ID**：格式类似 `cli_a1b2c3d4e5f6g7h8`
- **App Secret**：点「显示」后能看到一串字符

**两个都记录下来**：
```
我的 App ID:      cli________________________________
我的 App Secret:  ________________________________________________
```

### 4.4 添加机器人能力

1. 左侧菜单点击 **「添加应用能力」**
2. 找到 **「机器人」**，点击 **「添加」**
3. 添加后左侧菜单会多出「机器人」选项

### 4.5 配置事件订阅（关键！）

这一步让机器人能收到你@它的消息。

1. 左侧菜单点击 **「事件订阅」**
2. **请求方式** 选择 **「长连接」**（重要！这样不需要公网服务器）
   - 如果看到「使用长连接接收事件」的开关，打开它
   - 如果提示需要先配置加密策略，选「不加密」即可
3. **添加事件**：
   - 点击「添加事件」
   - 搜索 `im.message.receive_v1`（接收消息）
   - 勾选并确认添加

### 4.6 配置权限

1. 左侧菜单点击 **「权限管理」**
2. 搜索并开启以下权限（点右侧开关打开）：
   - `im:message`（获取与发送单聊、群组消息）
   - `im:message:send_as_bot`（以应用身份发消息）
   - `im:chat`（获取群组信息）

### 4.7 发布应用

1. 左侧菜单点击 **「版本管理与发布」**
2. 点击 **「创建版本」**
3. 填写版本号 `1.0.0`，可用范围选「所有员工」或「部分成员」（自己必选）
4. 提交发布

> 💡 如果你是企业管理员，可以秒批；如果不是，需要等管理员审批（一般几分钟内）。
> 个人测试空间（开发者空间）通常可以自批。

### 4.8 把机器人加到群聊

1. 打开飞书客户端
2. 创建一个测试群（或用现有群），把「市场早报助手」机器人添加进去
   - 群设置 → 群机器人 → 添加机器人 → 找到「市场早报助手」
3. 在群里 @机器人 发一条消息，比如 `@市场早报助手 你好`
   - 这一步是为了触发事件，让我们能拿到 chat_id

> ⚠️ 这一步只是为了让飞书记录这个群，等下启动服务后才能真正回复。

## 阶段 ⑤：填写配置文件 .env

### 5.1 创建 .env 文件

打开项目目录：
```
C:\Users\11195340\AppData\Roaming\BlueworkUi\bluework-temp-workspaces\e5bfd311\stock-morning-agent\
```

把 `.env.example` 复制一份，重命名为 `.env`（注意前面有个点）。

**操作方法**：
1. 在文件夹里右键 `.env.example` → 复制
2. 粘贴到同一文件夹
3. 把副本重命名为 `.env`（如果系统提示"必须键入文件名"，就先命名 `env`，再用 cmd 改名：`ren env .env`）

### 5.2 编辑 .env 文件

用记事本（或 VSCode）打开 `.env`，按下面格式填写：

```ini
# ========== LLM 配置 ==========
DEEPSEEK_API_KEY=sk-你刚才申请的真实Key
LLM_BASE_URL=https://api.deepseek.com
LLM_MODEL=deepseek-chat
LLM_TEMPERATURE=0.3

# ========== 飞书机器人配置 ==========
FEISHU_APP_ID=cli_你的AppID
FEISHU_APP_SECRET=你的AppSecret

# ========== 推送配置 ==========
PUSH_HOUR=7
PUSH_MINUTE=30

# 第一运行先随便填，启动后会从日志看到真实的 chat_id
FEISHU_RECEIVE_ID=先留空或随便填
FEISHU_RECEIVE_ID_TYPE=chat

# ========== 日志配置 ==========
LOG_LEVEL=INFO
```

**保存文件**（注意保存为 UTF-8 编码，记事本默认就是）。

### 5.3 验证配置

打开 cmd，cd 到 src 目录：

```cmd
cd /d "C:\Users\11195340\AppData\Roaming\BlueworkUi\bluework-temp-workspaces\e5bfd311\stock-morning-agent\src"
```

运行：

```cmd
python config.py
```

- 如果显示 `✅ 配置校验通过` → 继续下一步
- 如果显示缺少某些项 → 检查 .env 是否填对了

### 6.4 获取真实的 chat_id（如果第一次没填对）

如果 6.3 失败，提示 `未配置 FEISHU_RECEIVE_ID` 或 ID 错误：

1. 先在 .env 里把 `FEISHU_RECEIVE_ID` 填一个占位符，比如 `oc_test`
2. 运行 `python main.py serve` 启动服务
3. 在飞书群里 @机器人 发一条消息
4. 看 cmd 窗口的日志，会打印出类似：`收到消息 from xxx in oc_xxxxxxx`
5. 把 `oc_xxxxxxx` 这串字符填回 .env 的 `FEISHU_RECEIVE_ID`
6. 重启服务

## 🎉 完成！常见问题排查

### Q1: `pip install` 很慢或失败
**A**: 用国内镜像源加速：
```cmd
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q2: `python main.py fetch` 报错 "yfinance 无数据"
**A**: yfinance 访问雅虎财经需要网络能通到国外。如果公司网络限制，可以：
1. 开手机热点试试
2. 或配代理：在 .env 里加 `HTTP_PROXY=http://127.0.0.1:7890`（改你的代理端口）

### Q3: 飞书机器人收不到消息
**A**: 检查以下几点：
1. 确认 4.5 步骤选了「长连接」模式
2. 确认 4.6 步骤权限都开了
3. 确认 4.7 步骤应用已发布并审批通过
4. 确认机器人已加到群里

### Q4: 飞书能收消息但不回复
**A**: 检查日志（cmd 窗口或 logs\ 目录），看是否有报错。常见原因：
- DeepSeek API Key 错误 → 重新检查 .env
- 网络访问不了 DeepSeek → 测试 `ping api.deepseek.com`

### Q5: 控制台中文显示乱码
**A**: 这是 Windows cmd 的编码问题，不影响功能。如果想看正常中文，运行前执行：
```cmd
chcp 65001
```

### Q6: 想修改推送时间
**A**: 编辑 .env 文件，改 `PUSH_HOUR` 和 `PUSH_MINUTE`，重启服务。

### Q7: 想加更多监控标的
**A**: 编辑 `src\config.py`，在 `WATCH_SYMBOLS` 里加。标的代码去 https://finance.yahoo.com/ 搜。

### Q8: 怎么让程序一直运行（关电脑也不停）
**A**: 两种方式：
1. **最简单**：电脑一直开着，cmd 窗口不要关
2. **更专业**：部署到云服务器（阿里云/腾讯云轻量级服务器，9.9元/月），用 `nohup python main.py serve &` 后台运行
