# 📊 股票国际市场早报 Agent — 产品设计文档

> 作者：邹尔舒 | 2026年7月 | 独立开发项目

## 一、产品定位

一句话介绍：每天早上7:30自动推送到飞书的AI全球市场早报，不是资讯搬运，而是"新闻→影响链路→A股研判"的智能分析。

### 解决的问题

| 痛点 | 现状 | 本产品方案 |
|------|------|-----------|
| 信息过载 | 每天数百条财经新闻，散户无法高效消化 | AI自动筛选5-8条最重要新闻，并做影响链路分析 |
| 分析门槛高 | "美联储加息"对A股到底有什么影响？普通人看不懂 | 每条新闻附带"事件→传导路径→板块影响"的完整链路 |
| 缺乏个性化 | 所有财经App千人一面 | 用户偏好系统：关注AI/半导体就多分析科技新闻 |
| 推送渠道割裂 | 看行情要开App，看新闻要刷微博 | 飞书一站式推送+对话，@机器人即可提问 |

### 目标用户

| 用户画像 | 核心需求 | 使用场景 |
|---------|---------|---------|
| 散户投资者 | 快速了解隔夜全球市场+今日A股策略 | 每天早上看飞书早报 |
| 金融从业者 | 高效信息筛选，节省阅读时间 | 通勤路上看早报，@机器人追问 |
| AI产品经理（我自己） | 展示AI产品从0到1的全流程能力 | 作品集核心项目 |

## 二、核心功能

### 功能矩阵

- 行情监控：15个全球标的（美股/外汇/商品/加密/中概ETF）
- AI研判：新闻筛选 → 影响链路分析 → 情绪判断 → A股研判
- 定时推送：每日7:30自动推送到飞书群
- 对话交互：飞书@机器人随时提问
- 个性化：偏好配置 + Prompt注入 + 对话记忆

### AI研判体系（三级分析）

| 层级 | 分析内容 | AI能力 |
|------|---------|---------|
| L1 行情综述 | 隔夜全球市场全景概括 | 宏观策略分析师persona |
| L2 新闻影响链路 | 筛选5-8条最重要新闻，每条附带"事件→传导路径→A股板块影响" | 产业链传导分析 |
| L3 综合研判 | 今日A股情绪判断+大盘走势预判+板块建议+操作策略 | 策略分析师persona |

影响链路示例：
- 新闻：美联储7月会议纪要偏鹰
- 传导：维持高利率信号 → 美债收益率上行 → 全球风险资产估值承压
- A股影响：成长股短期承压，利好高股息防御板块
- 受影响板块：半导体、新能源、医药
- 情绪标签：利空

### 个性化系统（核心差异化）

| 能力 | 实现方式 | 效果 |
|------|---------|------|
| 偏好配置 | profiles/*.json 手动定义 | 关注板块、投资风格、风险偏好 |
| Prompt注入 | 偏好→AI system prompt | 每次生成早报/回答时自动匹配用户偏好 |
| 对话记忆 | SQLite记录所有对话 | 分析用户提问模式，自动识别兴趣板块 |
| 自动更新 | 定期分析对话历史 | 用户常问半导体→自动加入focus_sectors |

## 三、产品设计思路

### 为什么选择飞书而不是独立App

| 维度 | 飞书机器人 | 独立App |
|------|-----------|---------|
| 开发成本 | 0元，2天完成 | 数万元，数月开发 |
| 用户获取 | 飞书群一键添加 | 需要下载安装 |
| 推送触达 | 原生消息通知 | 需要用户打开App |
| 对话交互 | 原生IM体验 | 需要自建聊天UI |

### 为什么用DeepSeek

| 模型 | 价格（百万token） | 中文能力 | 推理能力 | 国内直连 |
|------|------------------|---------|---------|---------|
| DeepSeek | 输入1元/输出2元 | 优秀 | 优秀 | 是 |
| GPT-4o | 输入$2.5/输出$10 | 一般 | 优秀 | 否（需代理） |
| 智谱GLM-4 | 输入0.1元/输出0.1元 | 优秀 | 一般 | 是 |

### 个性化系统的设计原则

1. 渐进式个性化：新用户用默认配置，越用越懂你
2. 可解释性：用户可以随时查看和修改自己的偏好配置
3. 不牺牲客观性：prompt中明确要求"参考偏好但保持客观专业"
4. 隐私保护：每个用户的对话记录和偏好独立存储

## 四、数据指标

| 指标 | 数值 |
|------|------|
| 覆盖标的 | 15个 |

| 月成本 | 约16元（含对话） |
| 推送延迟 | 小于2分钟 |

## 六、个人收获

这个项目让我完整实践了AI产品从0到1的全流程：

1. 用户洞察：从自己和家人的真实需求出发，定义"不是资讯搬运而是AI研判"的产品定位
2. 技术落地：独立完成数据采集→AI分析→推送交付的全链路开发
3. Agent设计：将"影响链路分析"抽象为可复用的prompt工程范式
4. 个性化系统：设计了"偏好配置 + Prompt注入 + 对话记忆"的三层架构
5. 成本意识：在效果和成本之间做trade-off，选择DeepSeek实现日均0.03元的极致性价比
